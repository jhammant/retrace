# Screenshots

Run `uv run retrace serve` and open <http://127.0.0.1:8765> to see the live UI. Drop
PNGs here to illustrate the README. Suggested set:

- `now.png` — the **Now** view: the latest capture framed as a film cell with corner
  registration ticks, a Foundation Models caption, and app/source chips.
- `timeline.png` — the **Timeline**: a recorder spine with mono time ticks, app
  avatars, serif captions; click a row to expand full text + thumbnail.
- `search.png` — **Search** with the text / semantic / hybrid toggle and
  ember-highlighted matches.
- `stats.png` — the **Stats** dashboard: time-by-app (ember bars) and top domains
  (cyan bars), AFK-aware totals.
- `settings.png` — **Settings**: capture toggle, "Pause when away", sensitive-content
  controls, denylist editor, and the live permission grid.

The design language is a "nocturnal archive / flight-recorder console": warm
near-black, a single ember accent, editorial serif (Apple's *New York*) paired with a
forensic monospace for the recorder readouts.
